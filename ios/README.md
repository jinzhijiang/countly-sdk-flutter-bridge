#### Initalize iOS SDK or Update Version

Run init-ios-sdk.sh script to manage iOS SDK versions. 

```bash
./scripts/init-ios-sdk.sh
```

If it gives permission error

```bash
chmod +x scripts/init-ios-sdk.sh
```

Then try to run again.

To change the version, it is recommended to use VS Code’s source control tool. Countly iOS SDK appears there.
